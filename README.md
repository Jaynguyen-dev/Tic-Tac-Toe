## ğŸ“ Overview

This project is a **classic Tic-Tac-Toe game** developed in **C++**, using the  
**SFML (Simple and Fast Multimedia Library)** to render 2D graphics and handle user input.

The goal of this project is to practice:
- Core game logic implementation
- Event handling and rendering with SFML
- Structuring a small-scale C++ application
